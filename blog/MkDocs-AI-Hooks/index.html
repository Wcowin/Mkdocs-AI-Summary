    💾 AI Summary (ChatGPT)

    The MkDocs AI Hooks project enhances technical documentation and blogs with AI-driven summary generation and intelligent reading statistics using MkDocs hooks. Key features include AI-generated summaries, multi-language support, multiple API services, intelligent content cleaning, efficient caching, and flexible configuration. It also offers intelligent reading statistics with precise Chinese character counting, code detection, estimated reading time, and aesthetic information display. The tool provides automatic language recognition, content type detection, cache optimization, error handling, and advanced API integration for various AI services.

    MkDocs AI Hooks¶

    中文 | English

    🚀 您的MkDocs文档首选智能摘要！
    这个项目利用MkDocs hooks，为您的技术文档和博客添加AI驱动的摘要生成和智能阅读统计功能。

    网站效果预览：https://wcowin.work/Mkdocs-Wcowin/blog/Mkdocs/mkfirst/

    ✨ 功能特性¶

    AI智能摘要¶

    • 自动生成文章摘要：使用DeepSeek API生成高质量的80-120字摘要
    • 多语言支持：支持中文、英文等多种语言
    • 多API服务支持：支持OpenAI、Claude等多种AI服务
    • 智能内容清理：自动过滤YAML、HTML、代码块等格式内容
    • 备用摘要机制：API失败时提供基于规则的智能摘要
    • 高效缓存系统：避免重复API调用，7天智能过期
    • 灵活配置：支持文件夹级别和页面级别的精确控制

    智能阅读统计(可选)¶

    • 精准中文字符统计：专门优化的中文内容识别
    • 智能代码检测：识别多种编程语言和命令行代码
    • 阅读时间估算：基于中文阅读习惯的400字/分钟计算
    • 美观信息展示：使用MkDocs Material风格的信息框

    智能化特性¶

    • 自动语言识别：支持30+编程语言和标记语言
    • 内容类型检测：区分代码、配置、命令行等不同内容
    • 缓存优化：LRU缓存提升性能
    • 错误处理：完善的异常处理和日志记录

    📦 安装¶

    方法1：直接下载（推荐）¶

    在releases页面下载，解压后将以下文件放入您的MkDocs项目的docs/overrides/hooks中： https://github.com/Wcowin/mkdocs-ai-hooks/releases

    或者下载上方hooks目录下的两个Python文件： - ai_summary.py：AI摘要生成器 - reading_time.py：阅读时间统计器

mv reading_time.py docs/overrides/hooks/
